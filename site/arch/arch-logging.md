# Logging Patterns

This section defines the standard logging patterns used across all sheep-dog Java projects.

## Implementation Examples

For code examples and patterns, see the implementation files:

- [impl-slf4j.md](impl-slf4j.md) - SLF4J projects (parameterized logging with `{}` placeholders)

- [impl-vscode-outputchannel.md](impl-vscode-outputchannel.md) - VS Code extension projects (OutputChannel API)
