= Test-Suite: Code Generation for Only Issues

@sheep-dog-test

These tests verify that code generation works correctly when names follow proper capitalization rules.

== Test-Case: Cell name should start with a capital letter generation

* Given: The spec-prj project src/test/resources/asciidoc/specs/Process2.asciidoc file TestSuiteList/1/TestCaseList/1/TestStepList node is created as follows
+
|===
| Test Step Full Name                     
| The daily batchjob Input file is present
|===

* And: The Process2.asciidoc file TestSuiteList/1/TestCaseList/1/TestStepList/1/Table/RowList/1/CellList node is created as follows
+
|===
| Cell Name       
| lowercase header
|===

* And: The xtext plugin list quickfixes popup is set as follows
+
|===
| Proposal Id          | Proposal Description                    | Proposal Value  
| Capitalize cell name | Capitalize the first letter of the name | Lowercase header
|===

* When: The xtext plugin apply quickfix action is performed as follows
+
|===
| Node Path                                           
| TestSuiteList/1/TestCaseList/1/TestStepList/1/Table/RowList/1/CellList/1
|===

* Then: The spec-prj project src/test/resources/asciidoc/specs/Process2.asciidoc file TestSuiteList/1/TestCaseList/1/TestStepList/1/Table/RowList/1/CellList node will be created as follows
+
|===
| Cell Name       
| Lowercase header
|===
